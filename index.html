WJGA State Championship Pairer

How do I use this?
  1. Create a spreadsheet with the following columns:
    Column Description Possible values
    Name The player's name Any name
    District The player's district Numbers from 1 to 6
    Ranking The player's place at their Districts Championship. Give players who tied the same ranking. Give exempt players a ranking of 0 Numbers greater than or equal to 0
    Division (optional) The player gender/age division. If all players in the spreadsheet are in the same division, omit this column Boys 8-11, Boys 12-13, Boys 14-15, Boys 16-18, Girls 8-11, Girls 12-13, Girls 14-15, or Girls 16-18
    Examples: Spreadsheet with Division column Spreadsheet without Division column
  2. Download the spreadsheet as a .csv file.
  3. Upload that .csv file with the file picker below and hit Generate pairings.
